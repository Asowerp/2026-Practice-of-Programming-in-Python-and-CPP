# 魔兽世界大作业辅助工具使用指南

本工具覆盖 C++ 魔兽作业全流程：类设计、内存查看、游戏模拟、校验、日志对比。

## 1. 类编辑器
可视化设计 C++ 类结构，实时生成代码。
* 点击「添加类」新建类
* 填写类名、选择基类、添加成员变量
* 右侧实时预览生成的 C++ 代码
* 完成后点击「导出代码」

支持拖拽排序成员、右键删类、自动识别已有基类。

## 2. 内存视图
可视化查看类对象内存布局。
* 下拉选择对应类，查看内存条形图
* 悬停查看成员详细信息
* 红色：虚表指针 ｜ 蓝色：普通成员

可查看内存大小、理解内存对齐、验证继承内存布局。

## 3. 时间轴
回放整局游戏所有事件。
* 先在 Task2 运行模拟对局
* 进入时间轴查看完整事件列表
* 播放/暂停、步进、滑块跳转时间

查看兵力出生、交战、城池攻占节点。

## 4. Task1：武士层级校验
自动校验类继承结构合规性。
* 编辑器设计 Warrior 及其子类 (Dragon/Ninja/Iceman)
* 补齐 hp、attack 等必需成员
* 点击开始校验，根据报错修改代码

校验：继承关系、成员完整性、构造初始化列表。

## 5. Task2：整局模拟器
完整模拟标准对局，匹配 OJ 测试用例。
* 选用题目标准模式
* 自定义血量、出兵规则等参数
* 启动模拟查看对局结果

查看对局胜负、时序变化、兵力城池变动。

## 6. Task3：日志助手
比对输出与标准答案，快速排查格式 BUG。
* 导入 Task2 标准日志
* 粘贴自己程序输出进行比对

颜色标记：绿 = 正确 ｜ 红 = 错误 ｜ 黄 = 时间偏差，用于 OJ 提交预检。

## 快速上手流程
1. 类编辑器完成类结构
2. Task1 校验类设计
3. Task2 运行整局模拟
4. 时间轴复盘对局细节
5. Task3 日志比对后提交

├── main.py                      # 程序入口
├── constants.py                 # 常量定义
├── README.md                    # 用户文档
├── engine/                      # 业务逻辑层
│   ├── class_manager.py         # 类定义管理器
│   ├── models.py                # 数据模型
│   ├── battle_engine.py         # 战斗引擎
│   ├── warcraft_engine.py       # 魔兽游戏引擎
│   ├── task1_validator.py       # Task1校验逻辑
│   ├── task2_battle.py          # Task2战斗逻辑
│   ├── task3_log_helper.py      # Task3日志处理
│   └── timeline.py              # 时间轴控制
└── ui/                          # 界面层
    ├── mainwindow.py            # 主窗口
    ├── class_editor_widget.py   # 类编辑器界面
    ├── memory_view_widget.py    # 内存视图界面
    ├── task1_widget.py          # Task1界面
    ├── task2_widget.py          # Task2界面
